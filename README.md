# bookmarks

## User stories
 ```
As a user,
So that I can view a list of links ,
I'd like to use a tool to show the list of links.
```
```
As a User,
So that I can easily have access to the link I'd like to visit,
I'd like to save new links in a list.
```
```
As a User,
So that I can categorise my links ,
I'd like to add tags to each link.
```
```
As a User,
So that I can quickly find links from the category I am interested in,
I'd like to filter links by tags
